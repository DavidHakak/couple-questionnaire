# מדריך פריסה והרצה - שאלון זוגיות סינרגית

פרויקט זה מכיל שאלון אינטראקטיבי ומעוצב לזוגיות סינרגית, הכולל 45 שאלות המחולקות ל-10 קטגוריות, תמיכה בשמירה מקומית של טיוטות ל-48 שעות וייצוא ל-PDF בעברית (RTL).

## 🚀 הרצה מקומית

כדי להריץ את האתר במחשב המקומי:

1. ודא שיש לך **Python** מותקן במחשב.
2. פתח טרמינל בתיקיית השאלון והרע את הפקודה:
   ```bash
   python3 -m http.server 8080
   ```
3. פתח את הדפדפן בכתובת: `http://localhost:8080`

## 🌐 פריסה לאוויר (Vercel)

הפרויקט מוכן ב-100% לפריסה מיידית ב-Vercel. ישנן שתי דרכים פשוטות להעלות אותו לאוויר:

### דרך 1: באמצעות חיבור לחשבון GitHub (הדרך המומלצת 🌟)

זו הדרך הטובה ביותר מאחר וכל עדכון קוד ב-Git יעדכן אוטומטית את האתר באוויר:

1. **צור מאגר (Repository) חדש ב-GitHub**:
   - היכנס ל-GitHub וצור מאגר ריק (ללא README).
2. **העלה את הקבצים ל-GitHub מהמחשב שלך**:
   פתח טרמינל בתיקיית השאלון והרע את הפקודות הבאות:
   ```bash
   git init
   git add .
   git commit -m "initial commit"
   git branch -M main
   git remote add origin <URL_OF_YOUR_GITHUB_REPO>
   git push -u origin main
   ```
3. **חבר את המאגר ל-Vercel**:
   - היכנס לאתר [Vercel](https://vercel.com) והתחבר (למשל עם חשבון ה-GitHub שלך).
   - לחץ על **Add New** ואז על **Project**.
   - יבוא (Import) את המאגר שיצרת מ-GitHub.
   - לחץ על **Deploy**. האתר יעלה לאוויר תוך פחות מ-10 שניות ויקבל כתובת אינטרנט ציבורית!

### דרך 2: פריסה מהירה ישירות מהטרמינל (באמצעות Vercel CLI)

אם אינך רוצה להשתמש ב-GitHub, תוכל לפרוס ישירות מהמחשב שלך:

1. התקן את ממשק ה-CLI של Vercel (אם אינו מותקן עדיין):
   ```bash
   npm install -g vercel
   ```
2. בצע התחברות לחשבון ה-Vercel שלך:
   ```bash
   vercel login
   ```
3. פתח את טרמינל בתיקיית השאלון והרע:
   ```bash
   vercel
   ```
   *ענה על השאלות במסך (ניתן ללחוץ Enter לבחירת ברירות המחדל).*
4. כדי לפרוס סופית לפרודקשן (Production):
   ```bash
   vercel --prod
   ```

## 🛠️ טכנולוגיות בשימוש
- **HTML5 & CSS3**: עיצוב פרימיום נקי מותאם למובייל (Responsive).
- **JavaScript (Vanilla)**: ניהול שלבי השאלון והשמירה המקומית.
